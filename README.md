## 📱 About the App

Skillforge is a 3-screen Android application built in Kotlin that consumes the provided API to display categories, courses, and lessons in a clean, structured way.

---

## 🛠 Tech Stack

- **Language:** Kotlin
- **Architecture:** MVVM
- **Networking:** Retrofit + Coroutines
- **UI:** XML / Material Components
- **Async:** Kotlin Coroutines
- **State Management:** ViewModel + LiveData

---

## 📦 Features

- 📂 Categories screen
- 📚 Courses screen
- 🎓 Lessons screen with free/locked logic
- 🌐 API integration with the provided endpoint
- ⏳ Loading & error states
- ✅ Unit test for repository logic
